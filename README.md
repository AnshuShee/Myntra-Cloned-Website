# 🛍️ Myntra Clone (Frontend)

A responsive **Myntra website clone** built using **HTML** and **CSS**.  
This project replicates the UI of Myntra’s homepage with a focus on layout design, styling, and responsiveness.

---

## 🚀 Features

- 🎨 Modern and clean UI inspired by Myntra  
- 📱 Fully responsive design  
- 🧭 Navigation bar with categories  
- 🛒 Product listing section  
- 🔍 Search bar UI  
- 📦 Banner & promotional sections  
- 👣 Footer with useful links  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & Layout  
- **Flexbox / Grid** – Responsive design
